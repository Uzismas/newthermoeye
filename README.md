# Thermoeye

Thermoeye is a clinical screening console prototype for OCTA/OCT-based Alzheimer risk workflow support.

## Web Preview

The static preview is deployed from `web/` with GitHub Pages:

https://uzismas.github.io/thermoeye/

The Next.js application source lives in `app/` and `features/clinical-console/`.

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run typecheck
npm run build
```
